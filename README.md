# Interactive English Grammar Quiz
## Project Overview

This project is an interactive English grammar quiz designed for intermediate-level users.  
It allows users to answer multiple-choice grammar questions, receive immediate feedback, and track their score throughout the quiz.  
The aim of this project is to practise JavaScript interactivity by building a simple and responsive quiz application.
## User Goals

- Answer intermediate-level English grammar questions
- Receive immediate feedback on selected answers
- Track their score during the quiz
- View a final result at the end of the quiz
## Target Audience

- Adult learners of English
- Intermediate-level English speakers
- Users who want to test and refresh their grammar knowledge
- Students looking for a simple but engaging quiz experience.
